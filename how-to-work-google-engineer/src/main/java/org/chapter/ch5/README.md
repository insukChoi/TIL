## Chapter 5. 팀 이끌기

### 오늘날의 엔지니어링 관리자
- 전통적인 관리자는 일을 '어떻게' 처리할지를 고민하는 반면, 훌륭한 관리자는 '무슨' 일을 처리할지를 고민합니다.
- 팀원을 관리하려는 충동을 이겨내야한다. 관리가 아니라 겸손, 존중, 신뢰해주는게 리더의 자질.
